Detailed Description

Defines common attributes of the objects in the Asteroids game.

This class will act as a base class for CShip, CAsteroid, and CMissle classes. It contains the parameters such as the position, velocity, and lives of the object. It also contains methods to move the object, detect collisions, and draw it.
