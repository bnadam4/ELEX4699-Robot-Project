\doxysubsection{Detailed Description}
Implements an Etch-\/a-\/\+Sketch. 

This class defines the draw and update methods from the \mbox{\hyperlink{class_c_base4618}{CBase4618}} class so they can be used to implement an etch-\/a-\/sketch.
